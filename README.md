## Getting Started

_Follow the instructions below_

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/WilmerL2000/tiktok-clone-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API KEYS in `.env` file
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
4. Start project
   ```sh
   npm run dev
   ```
